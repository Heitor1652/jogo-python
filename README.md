# Jogo de Adivinhação em Python

Este projeto é um jogo onde o usuário tenta adivinhar
um número gerado aleatoriamente pelo computador.

# Funcionalidades:
- Gera um número entre 1 e 100
- Recebe palpites
- Dá dicas
- Conta tentativas

# Como executar:

1. Instale o Python
2. Baixe o arquivo
3. Execute no terminal:

python adivinhação. py

## Tecnologias:
- Python
- Biblioteca random
